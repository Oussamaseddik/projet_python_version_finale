Ce projet met en place une application Python permettant de gérer, analyser et rechercher des documents provenant de différentes sources (documents génériques, articles Arxiv et posts Reddit). Le code est organisé en plusieurs versions successives :

V1 introduit le socle de base avec les classes de documents, les auteurs, le corpus, ainsi que la sauvegarde et le chargement des données.

V2 ajoute un moteur de recherche simple basé sur TF, DF et TF-IDF, ainsi qu'une structure plus claire pour l’analyse textuelle du corpus.

V3 intègre une interface (Jupyter Notebook) permettant d’interagir facilement avec le corpus : ajout de documents, affichage, recherche, et visualisation des termes importants.
